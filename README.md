# 🎬 Movie Search App

A web application that allows users to search for and explore movies in real time using the OMDb API. The project provides a simple and intuitive interface to quickly find movie details.

---

## 🚀 Features

- 🔍 Search for movies by title  
- 🎞️ Display movie details (poster, year, type, etc.)  
- ⚡ Fast and dynamic data fetching using an external API  
- 📱 Responsive user interface  

---

## 🛠️ Technologies Used

- HTML5, CSS3, JavaScript  
- OMDb API (Open Movie Database)
